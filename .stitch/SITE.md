# FoundIt AI Website

## 1. Vision
A smart lost & found system for college campuses. It must be minimal, aesthetic, and modern with a clean layout, soft color palette, card-based design, and rounded components without unnecessary clutter. It connects students who lost items with finders, using a matching algorithm to notify the owner and security desk.

## 2. Sitemap
- [x] `index` (Landing / Login / Registration)
- [x] `dashboard` (Student overview: My Lost Items, Found Items submitted)
- [x] `report-lost` (Form: item name, description, color, location, image)
- [x] `upload-found` (Form: image, description, location found)
- [x] `matches` (Notifications & matched items list)
- [x] `admin` (Security desk management: all items, returns)
- [x] `analytics` (Admin charts & insights)
- [x] `qr-scan` (Mobile scanning interface for finders)

## 3. Roadmap
1. Build `index` landing page and auth flow.
2. Build `dashboard` for standard users.
3. Build `report-lost` and `upload-found` action forms.
4. Build `matches` screen.
5. Build `admin` and `analytics` for security.
6. Build `qr-scan` mobile flow.
